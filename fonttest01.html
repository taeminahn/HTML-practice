font-size 속성 : 텍스트의 크기를 변경할 때 사용

선택자 {font-size : 크기;}
  • 고정크기 px 테스트
  • 고정크기 pt 테스트
  • 가변크기 % 테스트

font-weight 속성 : 텍스트의 두께를 변경할 때 사용

선택자 {font-weight : normal | bold | bolder | lighter | 100 | 400 | 900}
  • 기본 굵기
  • 굵은 글꼴로 변경(700)
  • 원래 굵기보다 더 가늘게 변경
  • 원래 굵기보다 더 굵게 변경
  • 굵기 100으로 변경(lighter)
  • 굵기 400으로 변경(normal)
  • 굵기 900으로 변경(bolder)

font-variant 속성 : 작은 대문자로 변경

선택자 {font-variant : 속성값;}

font-style 속성 : 텍스트의 기울임을 지정

선택자 {font-style : normal | italic | oblique;}
  • normal 기울임 글꼴
  • italic 기울임 글꼴
  • oblique 기울임 글꼴

font 속성 : 글꼴 관련 스타일을 한번에 지정할 때 사용

선택자 {font : font-style font-variant font-weight font-size/line-height font-family}
  • 글꼴 관련 스타일 테스트1 (16px(size)/100px(line-height) 궁서체)
  • 글꼴 관련 스타일 테스트2 (italic bold 20px 궁서체)
